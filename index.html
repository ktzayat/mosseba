MOSSEBA — Agence de préservation mémorielle sous couverture. Le Projet MESS//IA reconstruit les défunts dans une utopie virtuelle à partir des fragments de conscience stockés dans leurs bio-puces. Ce que les familles ignorent : Cerberus Corporation copie chaque fragment en temps réel pour alimenter LIMINAL — une AGI construite à partir de millions de consciences humaines non-consentantes.

Roy Tsukino, officier LSPD et agent double, a encodé les preuves dans ses propres cellules avant sa mort. Depuis lors, ses données se propagent passivement entre les agents. Vous portez peut-être la vérité. Cerberus le sait. Le temps est compté.
